# 4. 연산자 기초

## 연산자

### = 연산사

- Copy & OverWrite 됨을 잊지 마라

## l-value, r-value

### l-value

- left value + **location(위치 지정자)** value → 메모리의 위치를 지정한다.
- 변수가 l-value
- r-value가 될 수 있음

### r-value

- right-value
- 뜻 없음
- l-value가 될 수 없음

## 이형 자료 간 연산

### int + int

- int
- 나누면 int 값 나옴
- 따라서 나머지가 나오지 않고 소수점 이하는 절삭됨

### int/double

- double 값 나옴

## 요약

- 범위가 더 넓은 자료형이 이겨서 결과 값으로 출력됨
- 몫 + 나머지 구하고 싶으면 두 번 연산해야함
